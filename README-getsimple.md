# GetSimple Project Cycle Interactive

This is a GetSimple theme conversion of `project-cycle-interactive.html`.

## Install

Copy these into your active GetSimple theme folder:

- `template.php`
- `asset/custom.css`
- `asset/project-cycle.js`

The template includes your existing `header.php` and `footer.php` if present.

## What is included

- Same inline SVG figure markup
- Same tooltip/card data
- Same embedded dummy PDF as base64 inside `project-cycle.js`
- Same pdf.js reader and fallback HTML reader
- Same CDN dependencies: Google Fonts and pdf.js

## Notes

Your header must output the normal page opening markup. Your footer must close it, as usual for a GetSimple theme.
